# Task Manager Application

## Overview
This is a simple full-stack Task Manager application built using React (frontend) and Node.js (backend). It allows users to add, view, and delete tasks.

## Features
- Add new tasks
- View list of tasks
- Delete tasks
- Simple and responsive UI

## Tech Stack
- Frontend: React
- Backend: Node.js (Express)
- Database: In-memory storage

## API Endpoints
- GET /tasks → Fetch all tasks
- POST /tasks → Add a new task
- DELETE /tasks/:id → Delete a task

## Setup Instructions
1. Clone the repository
2. Run backend:
   - node server.js
3. Run frontend:
   - cd client
   - npm start

## Approach
The application is built using a simple client-server architecture. React handles the UI and communicates with the Node.js backend via REST APIs.
